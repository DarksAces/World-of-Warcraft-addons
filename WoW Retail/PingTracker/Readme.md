# PingTracker

PingTracker is a lightweight World of Warcraft addon that monitors your latency in real time, alerts you to lag spikes, and stores zone-specific ping statistics. Keep track of your connection and be notified when latency exceeds a threshold.

## Features
- Displays current ping in a movable, resizable window.
- Tracks minimum, maximum, and average ping per zone.
- Alerts for lag spikes with a visual message and sound.
- Stores positional preferences for the ping window.
- Easy-to-use slash command for testing alerts.
- Updates automatically every 5 seconds.

## Installation
Download the addon from [CurseForge](https://www.curseforge.com/wow/addons/pingtracker).

## Usage

### /pingtest
Manually triggers a lag spike alert for testing purposes.

### Visual Ping Window
- Shows current ping in milliseconds.
- Drag to move the window; position is saved automatically.
- Color-coded text to easily see current ping.

### Example Output
`Ping: 42 ms`  
Lag spike detected: prints a red warning in chat and plays a sound.

# PingTracker

PingTracker es un addon ligero de World of Warcraft que monitoriza tu latencia en tiempo real, alerta sobre picos de lag y guarda estadísticas de ping por zona. Mantén un seguimiento de tu conexión y recibe notificaciones cuando la latencia supere un umbral.

## Características
- Muestra el ping actual en una ventana movible y redimensionable.
- Registra ping mínimo, máximo y promedio por zona.
- Alertas de picos de lag con mensaje visual y sonido.
- Guarda la posición de la ventana automáticamente.
- Comando slash sencillo para probar alertas.
- Actualización automática cada 5 segundos.

## Instalación
Descarga el addon desde [CurseForge](https://www.curseforge.com/wow/addons/pingtracker).

## Uso

### /pingtest
Activa manualmente una alerta de pico de lag para pruebas.

### Ventana de Ping
- Muestra el ping actual en milisegundos.
- Arrastra para mover la ventana; la posición se guarda automáticamente.
- Texto en color para identificar fácilmente el ping actual.

### Ejemplo de salida
`Ping: 42 ms`  
Pico de lag detectado: imprime una advertencia en rojo en el chat y reproduce un sonido.
