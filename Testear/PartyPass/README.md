# PartyPass

**Descripción:**
Automatiza la gestión de invitaciones sociales.

**Características:**
- **Auto Aceptar Grupo:** Acepta automáticamente invitaciones de grupo de amigos y amigos de Battle.net.
- **Auto Aceptar Invocación:** Acepta automáticamente las invocaciones de brujos o piedras de encuentro.

**Instalación:**
Copia la carpeta `PartyPass` a tu carpeta `Interface\AddOns`.
