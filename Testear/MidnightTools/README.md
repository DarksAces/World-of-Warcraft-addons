# MidnightTools

**Descripción:**
Este addon proporciona utilidades automáticas para facilitarte la vida en Azeroth.

**Características:**
- **Venta Automática:** Vende automáticamente todos los objetos grises (basura) cuando visitas a un comerciante.
- **Reparación Automática:** Repara automáticamente tu equipo si tienes suficiente oro al visitar a un comerciante.

**Instalación:**
Copia la carpeta `MidnightTools` a tu carpeta `Interface\AddOns`.
