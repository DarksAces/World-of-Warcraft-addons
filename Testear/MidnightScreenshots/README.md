# MidnightScreenshots

**Descripción:**
Captura automáticamente tus momentos de gloria.

**Características:**
- **Logros:** Toma una captura de pantalla automáticamente cuando ganas un logro.
- **Subida de Nivel:** Toma una captura de pantalla automáticamente cuando subes de nivel.

**Instalación:**
Copia la carpeta `MidnightScreenshots` a tu carpeta `Interface\AddOns`.
