# MidnightCinematics

**Descripción:**
Salta automáticamente las cinemáticas y escenas de vídeo.

**Características:**
- Cancela automáticamente las cinemáticas del juego.
- Cierra automáticamente las ventanas de reproducción de películas.

**Instalación:**
Copia la carpeta `MidnightCinematics` a tu carpeta `Interface\AddOns`.
