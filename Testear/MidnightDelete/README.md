# MidnightDelete

**Descripción:**
Facilita la eliminación de objetos importantes.

**Características:**
- Rellena automáticamente el texto "DELETE" (o su equivalente local si es necesario) en los cuadros de confirmación al intentar borrar objetos de calidad azul o superior.

**Instalación:**
Copia la carpeta `MidnightDelete` a tu carpeta `Interface\AddOns`.
