# MidnightKeystone

**Descripción:**
Automatiza la inserción de piedras angulares míticas.

**Características:**
- Coloca automáticamente tu Piedra Angular (Mythic Keystone) en la ranura cuando abres la fuente de poder en una mazmorra mítica.

**Instalación:**
Copia la carpeta `MidnightKeystone` a tu carpeta `Interface\AddOns`.
